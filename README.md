#  user-manager-cli-with-json-persistence 
A command-line interface (CLI) CRUD application for managing users with JSON file persistence. 
Users can be added, modified, or deleted, and the application uses key-value pairs for user attributes.

## Features:
- Interactive menu with numbered options (1, 2, 3, ...) to choose actions. 
- Yes/No confirmations (y/n) where needed.
- Tasks are stored in a json file (users.json) to keep data persistent.

## Main functions:
- Add users
- View all or one user
- Modify user properties
- Delete user

## How to user: 
- Clone the repository or download the files.
- Run the main program: menu.py
- Navigate the menu by entering the numbers of the options.
- For Yes/No prompts, type y for yes or n for no .
- Users are automatically saved in json_storage/users.json
